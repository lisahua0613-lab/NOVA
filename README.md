# Lisa Nova

A lightweight, beginner-friendly missile defense game built with **React**, **Tailwind CSS**, and **Canvas**.

## 🚀 Features
- **Simple Controls**: Tap/Click to intercept rockets.
- **Beginner Friendly**: Larger explosion radius and forgiving hit detection.
- **Beautiful Visuals**: Pink plum blossom background with smooth animations.
- **Responsive**: Works perfectly on Mobile, Tablet, and Desktop.
- **Bilingual**: Support for both Chinese and English.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite 6
- **Styling**: Tailwind CSS 4
- **Animations**: Motion (formerly Framer Motion)
- **Icons**: Lucide React
- **Rendering**: HTML5 Canvas

## 📦 Deployment to Vercel

1. **Push to GitHub**:
   - Create a new repository on GitHub.
   - Run `git init`, `git add .`, `git commit -m "Initial commit"`.
   - Add your GitHub remote and push: `git push -u origin main`.

2. **Connect to Vercel**:
   - Log in to [Vercel](https://vercel.com).
   - Click **New Project** and import your GitHub repository.
   - Vercel will automatically detect the **Vite** framework.
   - Click **Deploy**.

3. **Environment Variables**:
   - If you use the Gemini API features, add `GEMINI_API_KEY` in the Vercel project settings under **Environment Variables**.

## 💻 Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
